# Python Exercises

Small Python exercises created by Hossein Jordakani as part of a learning path toward cybersecurity scripting.

**Goal:** demonstrate problem-solving, clean logic, and basic scripting skills in Python.

## Contents
- `exercises/count_chars.py` — count frequency of characters in a string.
- `exercises/inventory_value.py` — calculate total value from a product dictionary.
- `exercises/generators_example.py` — examples of generators and memory-efficient processing.
- `exercises/filter_expensive.py` — example of generator-based filtering.

## Notes
These are learning exercises — not production tools. They show approach to break down problems, implement solutions, and document work.
